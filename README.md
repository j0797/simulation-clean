🌍 Интерактивная симуляция экосистемы, где хищники охотятся на травоядных, травоядные питаются травой, а вы наблюдаете за динамикой жизни в реальном времени.


🚀 Запуск

Требования: Java 17 или новее, Git (для клонирования репозитория), терминал с поддержкой Unicode(для отображения эмоджи)

1. Клонируйте репозиторий:
   git clone https://github.com/j0797/simulation-clean.git
   cd simulation-clean

2. Скомпилируйте проект:
   javac -d out -sourcepath src src/simulation/Main.java

3. Запустите симуляцию:
   java -cp out simulation.Main

🎮 Управление симуляцией - при запуске вы можете настроить параметры: размер мира - любой размер от 5x5 до 50x50, показ карты - отображение карты каждый ход или только при паузе, скорость - задержка между ходами (от 0 до 5000 мс). Команды во время работы: p - Пауза (приостановить симуляцию), r - Возобновить (продолжить с того же места), s - Статистика (показать текущий ход и статус), q - Выход (завершить симуляцию).

🏗️ Архитектура

Проект разработан в парадигме ООП с чётким разделением ответственности:

- **`Entity`** — базовый класс всех объектов
- **`Creature`, `Herbivore`, `Predator`** — существа с поведением и здоровьем
- **`WorldMap`** — хранит карту и размещение объектов
- **`BreadthFirstSearch`** — реализует алгоритм BFS для поиска пути
- **`Simulation`** — управляет ходами, паузой,остановкой
- **`ConsoleRenderer`** — визуализация состояния мира
- **`Action`** — паттерн Command: InitWorldAction - начальное заполнение мира, CreatureMoveAction - движение всех существ, GrassGrowthAction - рост новой травы

🎮 Возможности

- 🧭 **Поиск пути** реализован с помощью алгоритма BFS (поиск в ширину)
- 🐰 **Травоядные** едят траву и убегают от хищников
- 🐺 **Хищники** охотятся на травоядных
- 🌿 **Трава** появляется на карте в начале симуляции и добавляется каждый ход
- 🌳 **Деревья и камни** — статичные препятствия, блокирующие движение
- ⚙️ **Условия остановки** — симуляция автоматически завершается при: вымирании всех травоядных - хищникам нечего есть, вымирании всех хищников - теряется баланс экосистемы, вымирании всех существ - мир становится пустым. При остановке выводятся причина завершения, количество выполненных ходов, финальное состояние мира.
